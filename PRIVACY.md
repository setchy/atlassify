# Privacy Policy

Last updated: 2026-06-13

## Scope

This policy covers:

- The Atlassify desktop application (macOS, Windows, Linux)
- The Atlassify documentation website

## Desktop App Analytics

Atlassify uses [Aptabase][aptabase-website] for optional anonymous usage analytics in the desktop app.

### What We Collect

The desktop app may send anonymous product-usage events, such as:

- App lifecycle events (for example, app launched)
- Navigation events (for example, screen transitions)
- Feature and action usage (for example, refresh, mark read/unread, open links)

### What We Do Not Intentionally Collect

Atlassify does not intentionally send the following in analytics events:

- Atlassian API tokens
- Notification content or message bodies
- Account secrets

### Why We Collect It

We use anonymous analytics to:

- Understand which features are used most
- Identify UX friction
- Prioritize bug fixes and improvements

### Opt-Out

You can disable desktop app analytics at any time:

1. Open Settings
2. Go to System
3. Disable Share anonymous usage analytics

Disabling analytics does not disable core app functionality.

## Documentation Website Analytics

The Atlassify documentation website uses [Google Analytics][google-analytics-website] for website traffic and page-level usage insights.

## Data Processors

- Desktop app analytics: [Aptabase][aptabase-website]
- Website analytics: [Google Analytics][google-analytics-website]

## Data Retention

Analytics data retention and handling are subject to the configured retention policies of the analytics providers listed above.

## International Data Transfers

Analytics providers may process data in regions outside your country of residence.
